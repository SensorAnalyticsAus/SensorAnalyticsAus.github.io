This appliance is an Ubuntu virtual machine which runs the framework after start-up automatically. While the VA-App remains in Beta version, it will be provisioned free and supported. This trial version is not limited in any way, those participating in the Beta testing can retain this version, even after Beta trial period is over, for non-commercial use. Source code for the VA-App is also included in the vm.
